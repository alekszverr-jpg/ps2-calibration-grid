# PS2 Calibration Grid

Автономный генератор тестовых таблиц для PlayStation 2. После сборки получается
`PS2Grid.elf`; изображения и другие внешние файлы не нужны.

## Экраны

1. Геометрическая сетка 32 px, центральные оси и две окружности.
2. Рамки overscan: 100%, 95%, 90%, 85% и 80%.
3. Ступени серого и PLUGE-поля для настройки уровня чёрного.
4. Цветовые полосы 75%.
5. Точный по выходным пикселям checkerboard 16x16.
6. Сетка и перекрестия для проверки сведения лучей.
7. Вертикальные и горизонтальные полосы толщиной один и два пикселя для
   проверки резкости.
8. Цветовые границы для проверки растекания цвета и совмещения яркости с цветом.
9. Полноэкранные белая, серая, красная, зелёная и синяя заливки.

## Управление

- `Влево` / `Вправо` — предыдущий/следующий экран.
- `Крест` — следующий экран.
- `Квадрат` — открыть или закрыть меню таблиц.
- `Вверх` / `Вниз`, затем `Крест` — выбрать таблицу в меню.
- `Круг` — закрыть меню без смены таблицы.
- `Треугольник` — предпросмотр следующего видеорежима.
- `Крест` / `Круг` во время предпросмотра — оставить/вернуть режим.
- `Select` — показать/скрыть подсказку.
- `Start + Select` — выход в системный Browser PS2.

Начальный регион определяется автоматически по ROM консоли. PAL-консоли
запускаются в 576i/50, NTSC-консоли — в 480i/60.

Доступны PAL 576i, PAL 288p, NTSC 480i и NTSC 240p. `Треугольник` включает
предпросмотр следующего режима. Нажмите `Крест`, чтобы оставить его, или `Круг`,
чтобы вернуться. Без подтверждения предыдущий режим восстановится автоматически
через десять секунд.

## Запуск

Скопируйте `PS2Grid.elf` на USB-накопитель и откройте его через wLaunchELF/uLaunchELF.
Файл также можно добавить в Apps в OPL.

## Сборка

Нужны актуальные PS2SDK и gsKit. В окружении ps2dev:

```sh
export PS2DEV=/usr/local/ps2dev
export PS2SDK=$PS2DEV/ps2sdk
export GSKIT=$PS2DEV/gsKit
export PATH=$PATH:$PS2DEV/bin:$PS2DEV/ee/bin:$PS2DEV/iop/bin:$PS2DEV/dvp/bin:$PS2SDK/bin
make
```

Официальный полный контейнер ps2dev также подходит:

```sh
docker run --rm -v "$PWD:/src" -w /src ps2dev/ps2dev:latest make
```

Репозиторий также содержит GitHub Actions workflow. После каждого изменения в
ветке `main` он собирает и проверяет ELF, затем публикует архив `PS2Grid-ELF` в
разделе Actions → Artifacts. Сборку можно запустить вручную кнопкой Run workflow.

## Статус проверки

Версия 1.2 успешно проверена владельцем проекта на реальной PlayStation 2,
подключённой к кинескопному телевизору JVC AV-2130SE. Проверены все 13 таблиц,
меню быстрого выбора, режимы PAL 576i, PAL 288p, NTSC 480i и NTSC 240p,
автоматический и ручной безопасный откат, а также выход из программы.
Совместимость со всеми ревизиями консоли, типами кабелей и телевизорами не
гарантируется.

## Важное замечание

Не оставляйте статичную сетку надолго на плазменном или OLED-телевизоре.
Не оставляйте полноэкранную белую заливку надолго на кинескопном телевизоре.
Сначала запишите исходные значения сервисного меню телевизора; неверные
настройки геометрии могут сделать изображение недоступным.
